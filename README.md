# Nucleo

A submission for **[milkyway](https://milkyway.hackclub.com/)**!!

**Play at:** https://irtaza.itch.io/nucleo

## Description

**Nucleo** is a fast-paced endless wave survival game where you defend a atomic core from swarms of hostile particles.

Use **Alpha, Beta, and Gamma radiation** to destroy incoming enemies. Each radiation type has different **ionization power, range, and energy cost**, inspired by real nuclear physics.

Choose **1 of 3 upgrade cards** after each wave to strengthen your core.

## Controls

### Keyboard + Mouse

- Aim with the mouse

- Use radiation attacks to destroy incoming particles

- Choose upgrades between waves to improve your reactor

| Input          | Action               |
| -------------- | -------------------- |
| Mouse Movement | Aim                  |
| 1              | Fire Alpha Radiation |
| 2              | Fire Beta Radiation  |
| 3              | Fire Gamma Radiation |

## Gameplay

**Endless Waves:** Enemy particles continuously spawn and grow stronger over time.

**Radiation Types:** Alpha, Beta, and Gamma radiation each have unique strengths and weaknesses.

**Energy System:** Attacks consume reactor energy, which slowly regenerates.

**Upgrade Cards:** After each wave, pick 1 of 3 upgrades to enhance your build.

**Enemy Types:** Different particles resist different radiation types, encouraging strategic weapon use.
